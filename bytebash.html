Byte Bash is Here!

Get psyched, for today, we code! When: December 8th, 2015, 3:30 PM Where: District Meeting Room (next to the library) Who: Any MHS student or staff member, or sender school student or staff member
Why: To learn to code, create something with friends or have fun with computers and logic
  • Guest wi-fi access for personal machines
  • refreshments, pizza, and snacks provided
  • tutorials, contests, and prizes (oh, my!)
  • Be part of the international Hour of Code
Follow us: @MHSByte on Twitter Share the action from today's events using #ByteBash and/or #MundyPride

Today's Events

  • 3:30 - Welcome and Introductions
  • 3:45 - Tutorial Sessions
    • Code.org HOC Activities: Try these fun and easy coding tutorials. Get your feet wet and complete the hour of code!
    • Intro to Drawing with Khan Academy and ProcessingJS: Want to get in on the Character Contest? Learn to draw with code at Khan Academy's ProcessingJS module.
    • Codecademy tutorials: Looking to kick-start a serious study of coding? Codecademy has a fantastic series of tutorials for virtually any modern scripting or coding language. This is the place to start. Use this module to practice for the Web Page Contest as well
    • Kenny's Java circus!: Byte Club's President takes you through a basic project using the Java programming language.
    • Intro to FightCode: Start prepping for our FightCode competition! Learn to code a fighting robot that will take on your enemies!
  • 4:20 - Tutorial Showcase
    • How did you spend your first (almost) hour of code!?
    • Hello World Prizes
  • 4:30 - Coding Contests
    • Character Contest: Use Khan Academy and the drawing tools to create the best character possible
    • The Byte Club 5: Use any programming language you want to complete the Byte Club 5 Coding Challenges!
    • Web Site Contest: Code a website from scratch that has all the bells and whistles. Best site wins a prizes
      • Khan Academy's HTML Builder
      • CodePen.io
    • FightCode Duels: See how you rank at FightCode
    • Project Euler Battle: Form teams and take on the programming challenges at Project Euler. See who can complete the most in the time allotted
  • 5:30 - Contest Awards
  • 5:40 - Code with Byte Club
    • Take some time to explore programming and coding a little bit more. Talk with new and experienced members of Byte Club, show off your own projects and view those of others. See what we're all about.

You're Byte Club! Why doesn't this webpage look amazing!?!?

First, thanks for noticing. Second, as part of Byte Club we try to work as closely to the actual code as we can. Though there are many editors out there that would let us build a fantastic, multi-media, interactive webpage, we chose to keep it simple. Nearly all of the code on this page was created or adapted by Byte Club members or sponsors. No WYSIWYG editors are used. By the way, if you think you can make it look prettier - COME TO BYTE CLUB!
