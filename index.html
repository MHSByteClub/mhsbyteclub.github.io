Hello, World!

This is the homepage for the Mundelein High School programming club. We provide an environment for MHS students to program and learn to program. We're newly founded so there'll be more to see here later. (^̮^)

Goals for this Club

  • Give people a chance to learn and program with others

Projects we are currently doing or will be doing

Some things we'll definitely be working on:

  • Chrome extensions. MHS issues chromebooks to students which means these extension can have a big impact. For example, the school CISPA filter blocks a lot of relevant sites. Visiting all these sites to find that they're blocked is annoying and time-wasting, so a chrome extension that lets you know whether a link is blocked is bliss.
  • Processing sketches. For most new members byte club will be their first time coding and processing/processing.js is an ideal way to begin, as it allows you to write visual programs easily and immediately. You can make cool art, interactive demonstrations, and much more!
    • There's also a great curriculum and editor for it at Khanacademy.org.
  • Hackathons. This is when programmers get together and build cool things in a short span of time. Some (unaffiliated) examples of hackathons people could attend are CodeDay or any of the many listed at mlh.io.

Learning Sites

These are some websites to visit to learn to program

  • Codecademy • Offers great tutorials on Web development (HTML/CSS/Javascript/JQuery/Ruby/PHP) and Python.
  • KhanAcademy • Great editor for visual javascript programs. Also includes a full suite of videos to teach you how to program.
  • Bento • Includes lots of great links to programming tutorials.
  • CS50 • Harvard intro programming course. Teaches you computer science, languages used include include C, PHP, and JavaScript plus SQL, CSS, and HTML
  • Udacity • Offers a wide variety of Computer Science courses from beginner intro courses to advanced "holy cow we're making a self-driving car" courses. Includes really great introductory programming course on Python, although all of its courses are great.
